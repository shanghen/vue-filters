# vue-filters

> 基于vue2的简易filter(date,currency,number,lowercase,uppercase)  
PS:date 基于moment.js开发

## 安装和使用

``` bash
  npm install vue2-moment-filters --save

  import Filters from 'vue2-moment-filters'
  
  Vue.use(Filters,options)
```
